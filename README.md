# GoldenLeaf Farms 🌿

GoldenLeaf Farms is a full-stack farm fresh products shopping website where customers can select fruits, vegetables, and organic products through a live farm picking experience. The project includes customer ordering, checkout, order history, profile page, invoice download, and an admin dashboard to manage orders.

## Project Type

Python Full Stack Web Application

## Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Axios
- React Router DOM
- jsPDF

### Backend
- Python
- Django
- Django REST Framework
- SQLite Database

## Main Features

### Customer Side
- Home page with professional farm branding
- Farms category page
- Fruits farm
- Vegetables farm
- Organic products farm
- Live farmer picking animation
- Common basket for all farm categories
- Quantity increase/decrease
- Final bill summary
- Checkout form
- Payment method selection
- Saved customer details
- Order confirmation popup
- Customer profile page
- My Orders page using phone number
- Invoice download

### Admin Side
- Separate admin orders dashboard
- Password protected admin access
- View all orders
- Search and filter orders
- Dashboard cards
- Update order status
- Delete orders
- Export order data
- Download invoice
